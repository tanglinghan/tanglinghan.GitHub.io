Linux - 常用命令

2020-04-12 · 3 min read
Linux

查看当前时间

date

设置时区

cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

查看IP地址

ip addr

查看当前目录下各文件夹占用空间大小

du -h --max-depth=1 ./

查看 Linux 系统位数（备注：x86_64 表示 64 位系统， i686 i386 表示 32 位系统）

uname -a

将 /source/kernel.tgz 解压到 /source/linux-2.6.29 目录

tar -zxvf /source/kernel.tgz -C source/linux-2.6.29

防火墙相关

systemctl start firewalld
systemctl status firewalld 
systemctl stop firewalld

重启防火墙

firewall-cmd --reload

开放指定端口

firewall-cmd --zone=public --add-port=3306/tcp --permanent
firewall-cmd --zone=public --add-port=6379/tcp --permanent
firewall-cmd --zone=public --add-port=15672/tcp --permanent
firewall-cmd --zone=public --add-port=5672/tcp --permanent
firewall-cmd --zone=public --add-port=27017/tcp --permanent
firewall-cmd --zone=public --add-port=8081/tcp --permanent 
 命令含义：
--zone #作用域
--add-port=1935/tcp  #添加端口，格式为：端口/通讯协议
--permanent  #永久生效，没有此参数重启后失效

服务器之间文件传输

scp  -v -r [文件名称] root@[目标ip]:[目标服务器路径]
scp -v -r front root@10.10.66.154:/usr/local/bfp-admin-2.0-dev

查看磁盘信息

df -hT 
df -h
fdisk -l

查看已有硬盘及分区

lsblk

搜索进程

### 1. 根据端口号查进程 
pid  lsof -i:[port]
netstat -nap | grep [port]
### 2. 根据进程号PID查端口
netstat -nap | grep [pid]
### 3. 根据服务名查看进程 pid
ps -ef | grep java

查看服务安装路径

ll /proc/pid/cwd

后台启动java进程

nohup java -jar tabifcp-eureka-2.0.3.RELEASE.jar >./logs/tabifcp-eureka.log 2>&1 &
或者
nohup java -jar server-eureka-2.0.3.RELEASE.jar &>./logs/server-eureka.log &

批量杀死筛选进程

ps -ef | grep [关键字] | grep -v grep | awk '{print $2}' | xargs kill -9
